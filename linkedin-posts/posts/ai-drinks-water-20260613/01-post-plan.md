# Post Plan — ai-drinks-water-20260613

## Core Insight
Every "free" AI query hides a physical cost — water and energy — that never appears on any invoice, but is increasingly real in a water-scarce world.

## Target Audience
Entry-to-mid career software engineers who use AI tools daily without thinking about the infrastructure behind them — the same people who would optimize a SQL query for latency but never consider the data center cooling bill.

## Structure
1. The mundane action: user types a prompt and gets an answer in seconds — nothing feels costly
2. The invisible infrastructure: that exchange runs on servers generating heat, cooled by water — real, drinkable water
3. The irony: it's "free" on the invoice but has a physical cost the user never sees
4. The reflection: awareness isn't a call to stop using AI — it's a call to acknowledge what's actually behind the "Send" button

## CTA
"Do you think about what a single AI query actually costs — beyond the API bill?"

## Tone Range
Sarcastic, provocative, then grounded — open with dry wit, land with a real question
