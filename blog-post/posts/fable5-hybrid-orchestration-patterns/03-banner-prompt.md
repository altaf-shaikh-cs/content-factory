## Banner Image Prompt

**Core metaphor:** One large, bright node connected by thin glowing lines to several smaller nodes doing the actual work, evoking a single expensive resource directing a cheaper network instead of doing everything itself.

**Prompt (universal base):**
A dark, minimal editorial illustration. Deep charcoal background. One larger glowing node positioned above center, connected by thin glowing lines to several smaller nodes arranged below it, each smaller node pulsing faintly as if actively working. The large node is calm, still, sending only occasional signals down the lines. Negative space dominates the composition. Muted single accent color (amber or cyan) traces the connecting lines and the smaller nodes. Wide banner format, no text, no logos, no user interface elements, no code, no photorealism, flat conceptual editorial tech-blog illustration style.

**Tool modifiers:**
- Midjourney: append `--ar 16:9 --style raw --v 6`
- DALL·E / GPT-4o: add "digital illustration, flat design, no gradients"
- Ideogram: add "flat vector illustration, editorial style, tech blog"

**Do NOT include:** code on screens, company logos, text or titles, photorealism
